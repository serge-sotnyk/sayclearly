## SayClearly

Local-only diction training tool with a web UI and Gemini-based feedback.

The MVP specification lives in `docs/sayclearly_mvp_spec_en.md`.

### Setup

```bash
uv sync
```

### Run

```bash
uv run sayclearly
```

### Development

```bash
uv run pytest
uv run ruff check .
uv run ruff format --check .
```
